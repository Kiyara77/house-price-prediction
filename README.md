# House Price Prediction

A machine learning project for the [House Prices: Advanced Regression Techniques]( https://www.kaggle.com) competition.

- Performed data cleaning, feature engineering, and model tuning.
- Used models: XGBoost.
- Final submission achieved RMSE: 0.13764 on Kaggle public leaderboard.

## How to Run

1. Combine `train.csv` and `test.csv` to handle missing values jointly.
2. Import necessary libraries such as pandas and numpy.
3. Load the `house-price-prediction` code into Jupyter Notebook and execute.

This project is part of my graduate school application portfolio.
